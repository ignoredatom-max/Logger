# Logger 🎬📺🎮

A personal media diary PWA to track movies, series and games you've watched or played.

## Features
- Log films, series and games with ratings, status and notes
- Diary view grouped by month
- 5-stage rating system — Skip / Mediocre / Decent / Excellent / Peak
- Watchlist to queue things up
- Favourites page
- Google account sync via Firebase — data safe forever
- Installable as a PWA on Android home screen

## Built with
- Vanilla HTML / CSS / JS — single file, no framework
- Firebase Auth + Firestore for cloud sync
- TMDB API for movies & series data
- RAWG API for games data

## Install on Android
1. Open the GitHub Pages link in Chrome or Brave
2. Tap the 3-dot menu → Add to Home Screen
3. Sign in with Google to sync your data
